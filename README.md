# Shaan Rais Client Dashboard

## Project Info
This project is an administrative/client dashboard, built using modern web technologies.

## Technologies Used
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

To run this project locally, make sure you have Node.js installed.

1. Clone this repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```
4. Build for production:
   ```sh
   npm run build
   ```
